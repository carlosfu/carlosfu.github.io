Hbase学习1.1-单机版安装

发表于 2016-11-28   |   分类于 hbase   |     |   阅读次数

注意

1. 需要oracle jdk 7+。
2. Hbase0.98分别支持hadoop1和hadoop2，hadoop1已经不更新了，所以选用hbase0.98-hadoop-2。
3. 单机版hbase: hbaseMaster,zookeeper,regionserver都在一起，使用本地磁盘代替hdfs存储。

一、安装、配置、启动

1.下载

hbase镜像：http://mirror.bit.edu.cn/apache/hbase/

3. 修改Hbase相关环境变量
4. 修改hbase-site.xml配置

如果不添加hbase会安装在/tmp/hbase-${user.name}目录下，在机器重启时候会删掉

5. 修改hbase-env.sh中的JAVA_HOME地址
6. 启动
