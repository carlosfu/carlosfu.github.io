jvm学习之-jstat

发表于 2016-06-19   |   分类于 jvm   |     |   阅读次数

一. 简单说明

jstat是用于监控虚拟机各种运行状态信息的命令工具，尤其对于了解垃圾回收的情况很有帮助(类装载、垃圾收集、运行期编译状况)。

二. 使用方法

下图来自深入理解java虚拟机，这里只选几个重要的进行说明
